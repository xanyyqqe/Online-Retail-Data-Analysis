# Анализ данных Online Retail
Этот проект посвящен анализу данных онлайн-ритейла с использованием ключевых бизнес-метрик: **DAU (Daily Active Users), MAU (Monthly Active Users), Stickiness, Interval Retention и ARPU (Average Revenue Per User).**

## Используемые технологии

SQL (SQLite3) – для работы с базой данных.

Python 3 – основной язык программирования.

Pandas, NumPy – для работы с данными и их обработки.

Matplotlib, Seaborn – для визуализации данных.

## Структура проекта

**analysis_docs/** – содержит отчетные выводы и рекомендации:

conclusions.md – ключевые выводы по анализу.

recommendations.md – рекомендации по удержанию пользователей и увеличению прибыльности бизнеса.

**sql/** – SQL-запросы, использованные в процессе анализа.
**graphs/** – основные графики, демонстрирующие закономерности по бизнес-метрикам.
**experiments.ipynb** – основной процесс аналитики (за исключением анализа в Excel).

## Цели проекта

* Анализ активности пользователей и их удержания.
* Оценка доходности бизнеса на основе метрик.
* Разработка рекомендаций для улучшения бизнес-показателей.
