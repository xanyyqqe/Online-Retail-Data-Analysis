# Рекомендации по удержанию и увеличению ценности пользователей

## 1. Новые пользователи (первые 30 дней)

* Сегментация по источнику привлечения.
* Персонализированные приветственные сообщения и акции.

## 2. Удержание и повторные визиты

* Триггерные уведомления для снижения оттока.
* Программы лояльности для увеличения повторных визитов.

## 3. Финансовая ценность

* A/B-тесты акций и ценовых стратегий.
* Апсейл/кросс-сейл и рекомендации товаров.

## 4. Фокус по рынкам

* Сфокусироваться на удержании пользователей в Великобритании
* Cтимулировать повторные заказы в странах с небольшой аудиторией.

## 5. Фокус по продукции

* Пересмотреть ассортиментную политику в пользу товаров с высокой маржинальностью.
* Рассмотреть возможность оптимизации или замены убыточных позиций; убыточные позиции сохранены в файле **unprofitable_stocks.csv**
